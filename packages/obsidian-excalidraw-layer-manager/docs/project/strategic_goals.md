# Strategic Goals

- Goal: Make Layer Manager X the strongest projection-based layer workspace available for Excalidraw-in-Obsidian without requiring native upstream layers.
  Metric: the package delivers a clear Hybrid Pro Panel experience over native scene structure while preserving deterministic mutation behavior and bounded performance on large scenes.

- Goal: Establish `customData.lmx` as the package-owned semantic memory layer for stable labels and future metadata.
  Metric: element labels and synthetic group labels survive common rename/reparent/regroup flows without corrupting foreign `customData` or pretending to be upstream canonical truth.

- Goal: Replace prompt-heavy power-user flows with an in-panel interaction model that feels faster, clearer, and more professional.
  Metric: common structure operations (rename, move, reorder, search, isolate, inspect) are primarily completed from the sidepanel surface rather than ad-hoc prompts.

- Goal: Evolve the panel from a layer tree into a high-trust scene operating surface only after the pro-panel foundation is excellent.
  Metric: later intelligence features improve orientation and cleanup without weakening determinism, responsiveness, or explainability.
