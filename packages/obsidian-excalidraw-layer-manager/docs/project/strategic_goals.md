# Strategic Goals

- Goal: Make Layer Manager X the strongest **high-trust projection-based** layer workspace available for Excalidraw-in-Obsidian without requiring native upstream layers.
  Metric: the package delivers a clear Hybrid Pro Panel experience over native scene structure while preserving deterministic mutation behavior, explainable derivation, and bounded performance on large scenes.

- Goal: Establish a projection kernel that keeps authority splits explicit instead of letting UX slices invent shadow structure.
  Metric: scene truth, metadata truth, structural truth, visible truth, and convenience truth remain distinct in both docs and code; cross-slice features consume shared contracts rather than local reinterpretations.

- Goal: Keep `customData.lmx` as the package-owned semantic memory layer for stable labels and bounded metadata only.
  Metric: labels survive common rename/reparent/regroup flows without corrupting foreign `customData` and without turning `customData.lmx` into a second structural engine.

- Goal: Replace prompt-heavy power-user flows with an in-panel interaction model that feels faster, clearer, and more professional.
  Metric: common structure operations (rename, move, reorder, search, isolate, inspect) are primarily completed from the sidepanel surface rather than ad-hoc prompts, while affordances remain honest about real scope and outcomes.

- Goal: Evolve the panel from a layer tree into a high-trust scene operating surface only after the projection kernel is excellent.
  Metric: later intelligence features improve orientation and cleanup without weakening determinism, responsiveness, or explainability.
