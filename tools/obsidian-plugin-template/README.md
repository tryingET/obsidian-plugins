# obsidian-plugin-template (seed)

This is an **internal seed template area**, not yet a standalone template product.

Purpose:
- capture the common package shape once the first real plugin packages stabilize
- avoid prematurely freezing bad package seams into an external template repo

Current rule:
- use this as a staging area for repeatable package contracts
- do **not** treat it as a publishable or cross-repo template yet

Graduation rule:
- extract a standalone template only after at least two real plugin packages prove the shared package shape
