# obsidian-excalidraw-script-template (seed)

This is an internal seed template area for Excalidraw-script-style packages.

It exists because `obsidian-excalidraw-layer-manager` is now the first real imported host package of that shape.

Current rule:
- use the imported package as the main proving ground
- treat this template as a seed only
- do not extract a standalone template repo yet
