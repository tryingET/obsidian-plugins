# lab-vault

This app folder is the local proving ground for the Obsidian plugin family.

It should hold:
- plugin dev fixtures
- sample notes / PDFs / images
- Excalidraw scenes for regression checks
- manual validation recipes

It is **not** canonical runtime or promotion authority.
It is a local host/workbench proving ground.
